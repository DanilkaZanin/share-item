# Фаза 3. Оценка функционального объёма и трудоёмкости проекта

## Назначение фазы
Фаза 3 посвящена комплексной оценке функционального объёма и трудоёмкости сервиса шеринга предметов с использованием формализованных методов. Цель — определить ориентировочные трудозатраты, сроки разработки и состав команды, а также сформировать основу для планирования MVP и календарно-ресурсного графика.

В ходе работы применялись четыре метода:

1. COSMIC Function Points — подсчёт функций и потоков данных.
2. Use Case Points — оценка сложности пользовательских сценариев.
3. Экспертная оценка — коллективное обсуждение и оценка Story Points.
4. User Stories — декомпозиция Use Cases на мелкие задачи с оценкой сложности.

| Метод | Результаты |
|-------|------------|
| COSMIC Function Points | ~30–35 CFP |
| Use Case Points | ~90–110 UCP |

Минимальная оценка: 440 ч  
Максимальная оценка: 1490 ч  
Средняя реалистичная оценка: ~850 ч

**Итоговое планирование:** 900 часов работы с запасом 15% на непредвиденные сложности.

---

## 2. MVP проекта

**Функциональность, которая войдёт в первую версию:**

- регистрация и профиль пользователя;
- базовые рекомендации одежды;
- простая 2D-виртуальная примерка;
- возможность сохранять понравившиеся образы.

**Срок реализации MVP:** 3 месяца  
**Бюджет:** ~1,4 млн руб

**Что откладывается на будущее:**

- 3D-примерка с дополненной реальностью;
- интеграция с большим количеством магазинов;
- автоматическое обновление гардероба.

**Причина:** 80% пользы для пользователей дают 20% функций. Быстрый релиз MVP позволит получить обратную связь и постепенно развивать проект.
