# Фаза 5. Календарно-ресурсное планирование проекта

## 1. Назначение фазы

Цель пятой фазы — составление подробного календарного плана выполнения проекта, исходя из декомпозиции задач, оценок их трудоёмкости и распределения ответственности.

Результаты этой фазы позволяют:

- определить последовательность выполнения задач;
- оценить сроки реализации;
- обеспечить равномерную загрузку участников команды.

---

## 2. Методика планирования

Для планирования использовались следующие подходы:

- **Трёхточечная оценка задач (PERT)**:
    - O — оптимистичная оценка;
    - M — наиболее вероятная оценка;
    - P — пессимистичная оценка;  
      Математическое ожидание: `(O + 4M + P) / 6`.
- **Учет ресурсов**: задачи распределены между Ильёй и Данилом.
- **Учёт зависимостей**: задачи выполняются в логической последовательности.

Для упрощения расчёта использован **рабочий день 8 часов** и 40-часовая рабочая неделя.

## 6. Выводы по фазе

В рамках пятой фазы был построен детальный календарный план проекта, с учётом:

- трудоёмкости задач;
- логической последовательности;
- распределения задач между Ильёй и Данилом;
- соблюдения 8-часового рабочего дня.

План позволяет:

- визуализировать прогресс проекта;
- равномерно распределять нагрузку между участниками;
- оценивать сроки завершения разработки сервиса шеринга предметов.
